关于博客图片加载问题

##关于图片加载的问题

1.自昨天建立博客后,今天想加图片,尝试了很久，图片是到了仓库，可是加载不出来。
在网上一顿找办法，后来在一篇博客文章中找到了办法
地址如下：https://www.cnblogs.com/xiaohuiduan/p/9867636.html
2.明白了问题有两种
①图片路径出错
路径出错，因为仓库名是自己的博客名，但是_config.yml中的URL却是默认的Ericam.com，导致了加载失败。将URL改成仓库地址就可以解决
②插件中出现了bug，因为hexo版本的更新，所以插件index.js出现了bug,只需如下图

修改完后，再进行hexo clean hexo g hexo d 三件套就可成功

博客搭建历程

##使用node.js环境下的Hexo框架搭建博客

1.安装 Node.js 环境
2.安装Git工具
3.全局安装Hexo “npm install hexo-cli -g”过程省略
4.使用hexo deploy将项目部署到github上
5.每次更新文章使用hexo clean清理缓存,然后使用hexo g加载,最后再hexo deploy重新部署到githuub的仓库里就成功了。

win10磕磕撞撞研究了2个半小时终于成功。
